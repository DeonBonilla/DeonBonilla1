# 🖥️ Active Directory Home Lab

## 📋 Overview
Built a virtualized Active Directory environment to simulate common IT Support and Help Desk tasks performed in enterprise environments. This lab provided hands-on experience with domain administration, user management, authentication, Group Policy, and endpoint management using Windows Server and Active Directory Domain Services.

## 🛠️ Technologies Used
- Windows Server 2022
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy
- Oracle VirtualBox
- Command Prompt

## 🎯 Objectives
- Deploy Active Directory Domain Services
- Configure a Domain Controller
- Create and manage domain users
- Configure Organizational Units and Security Groups
- Join endpoints to the domain
- Verify domain authentication
- Implement password and account policies
